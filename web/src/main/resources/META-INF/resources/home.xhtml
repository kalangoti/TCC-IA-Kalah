VENCENDO O JOGO:

Como desenvolver uma plataforma que permita a implementação de redes neurais voltadas para o aprendizado do jogo Kalah?

Você tem a opção de treinar o algoritmo de rede neural artificial contra um dos algoritmos disponíveis, ou jogar livremente.

Na forma de treinamento existem três dificuldades, onde cada uma delas utiliza um algoritmo específico para treinar os pesos da rede neural.

Já no modo jogo pode escolher entre jogar contra os algoritmos ou contra outro jogador humano.
